# Install

```bash
pip install --pre backtest360-client
```

Requires Python 3.9+. Runtime dependencies: `httpx`, `pandas` — that's it.

## Supported Python versions

3.9, 3.10, 3.11, 3.12.

## Install from source

```bash
git clone https://github.com/Backtest360/backtest360-client.git
cd backtest360-client
pip install -e ".[dev]"
```
